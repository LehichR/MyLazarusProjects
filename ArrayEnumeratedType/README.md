**Array with Enumerated Types** is a sophisticated Delphi application demonstrating advanced data structure concepts. The program creates a mapping between two enumerated types (Russian surnames and given names), populates it randomly, and identifies duplicate associations using efficient search algorithms.

### üéØ Key Features
- **Enumerated Type Implementation**: Uses Pascal's strong typing with custom enumerated types
- **Random Array Generation**: Creates random associations between surname and name types
- **Duplicate Detection**: Efficient algorithm to find duplicate name associations

- **Interactive Controls**: Multiple buttons for different operations with clear feedback

### üîç Algorithm Highlights
#### **Duplicate Search Algorithm:**
```pascal

  for b := succ(a) to HITROV do
```

- **Efficient Nested Loops**: Uses `succ()` to avoid redundant comparisons
- **Optimized Search**: O(n¬≤/2) complexity instead of O(n¬≤)
- **Early Exit**: Breaks loops when first duplicate is found
